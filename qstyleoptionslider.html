Detailed Description

该QStyleOptionSlider类是用来描述所需要的绘制一个滑块的参数。

QStyleOptionSlider包含的所有信息，QStyle功能需要绘制QSlider和QScrollBar。

出于性能方面的原因，访问成员变量是直接的（即使用. or ->操作符）。这种低层次的感觉，使结构简单的使用，并强调这些只是所用的样式函数的参数。

举一个例子展示风格选择如何使用，请参阅Styles例子。

Constant Value Description
QStyleOptionSlider.Type SO_Slider 风格选择该类型提供（SO_Slider这个类） 。

该类型在内部使用QStyleOption，它的子类，并qstyleoption_cast（）来确定的样式选项的类型。一般来说，你不需要担心这个，除非你想创建自己的QStyleOption子类和你自己的风格。

See also StyleOptionVersion。

Member Documentation

bool dialWrapping

这个变量保存的拨号是否换行还是不行。

默认值为False ，即转盘没有包裹。

See also QDial.wrapping（ ） 。

int maximum

这个变量保存为滑块的最大值。

默认值是0。

int minimum

这个变量保存为滑块的最小值。

默认值是0。

float notchTarget

这个变量保存像素的缺口之间的数字。

默认值是0.0 。

See also QDial.notchTarget（ ） 。

Qt.Orientation orientation

这个变量保存滑块的方向（水平或垂直） 。

默认的方向是Qt.Horizontal。

See also Qt.Orientation。

int pageStep

这个变量保存滑块的页面步长的大小。

默认值是0。

See also QAbstractSlider.pageStep。

int singleStep

这个变量保存滑块的单一步骤的大小。

默认值是0。

See also QAbstractSlider.singleStep。

int sliderPosition

这个变量保存滑块手柄的位置。

如果滑块具有积极的反馈（即，QAbstractSlider.tracking为真） ，则此值将是相同的sliderValue。否则，它将具有手柄的当前位置。默认值是0。

See also QAbstractSlider.tracking和sliderValue。

int sliderValue

这个变量保存滑块的值。

如果滑块具有积极的反馈（即，QAbstractSlider.tracking为真） ，则此值将是相同的sliderPosition。否则，它的值将鼠标被按下之前，滑块了。

默认值是0。

See also QAbstractSlider.tracking和sliderPosition。

int tickInterval

这个变量保存，应该刻度线之间绘制的时间间隔。

默认值是0。

QSlider.TickPosition tickPosition

这个变量保存滑块的刻度线的位置，如果有的话。

缺省值是QSlider.NoTicks。

See also QSlider.TickPosition。

bool upsideDown

这个变量保存滑块控件方向。

通常一个滑块随着它向上移动或向右侧; upsideDown表明它应该做相反（增加，因为它移动向下或向左） 。默认值为False，即滑块随着它向上移动或向右移动。

See also QStyle.sliderPositionFromValue（ ）QStyle.sliderValueFromPosition（）和QAbstractSlider.invertedAppearance。
