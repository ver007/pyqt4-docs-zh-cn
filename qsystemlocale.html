Detailed Description

The QSystemLocale class can be used to finetune the system locale of the user.

Warning:这个类只有在非常罕见的情况下非常有用。平时QLocale提供了所有必需的应用程序开发的功能。

QSystemLocale允许复盖系统区域设置（提供的值QLocale.system（））。
