Detailed Description

该QStringListModel类提供了一个模型，它提供字符串的看法。

QStringListModel是可用于在您需要显示的字符串有多个视图中的小部件，如一个简单的情况下可编辑模式QListView或QComboBox。

该模型提供了一个可编辑的模型中所有的标准功能，即在字符串列表与一个列和一个行数等于项目的列表中的号码的一个模型中的数据。

对应的项目模型索引以获得index()功能和项目标志与获得flags（ ） 。项目数据的读取使用data（）函数和写入setData（ ） 。行（并在字符串列表项目数）的数量可以与被发现rowCount（）函数。

该模型可以与现有的字符串列表来构建或字符串可以在以后的设置setStringList（ ）的便利功能。串，也可以与通常的方式插入insertRows（）函数，并用除去removeRows（ ） 。字符串列表的内容可以检索与stringList（ ）的便利功能。

QStringListModel的使用示例：

     QStringListModel *model = new QStringListModel();
     QStringList list;
     list << "a" << "b" << "c";
     model->setStringList(list);
