Detailed Description

该QDomText类表示解析的XML文档中的文本数据。

您可以在两个QDomText objecs与分裂的QDomText对象中的文本splitText（ ） 。

关于文档对象模型有进一步的信息Level 1和Level 2 Core。对于更一般介绍的DOM实现的见QDomDocument文档。

QDomText QDomText.splitText (self, int offset)

这个分裂的DOM文本对象分为两QDomText对象。这个对象保持其第一offset字符和第二个（新建）对象被插入到文档树中剩馀的字符这个对象之后。
