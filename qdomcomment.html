Detailed Description

该QDomComment类表示一个XML注释。

在解析XML像这样的注释：

 <!-- this is a comment -->

通过在所解析的DOM树QDomComment对象表示。

关于文档对象模型有进一步的信息http://www.w3.org/TR/REC-DOM-Level-1/和http://www.w3.org/TR/DOM-Level-2-Core/。对于更一般介绍的DOM实现的见QDomDocument文档。
